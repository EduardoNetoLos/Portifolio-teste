Projetos

Guia do Aluno

O Guia do Aluno é um projeto focado em ajudar alunos com problemas de organização a estudarem de forma eficiênte. Essa aplicação possui suporte para Web e Android. Ela conta com funcionalidades como uma agenda para que você programe suas atividades pendentes, uma área com as datas das provas para que você se mantenha antenado nas datas dos vestibulares, olimpíadas e do ENEM, além de dicasa de como estudar de forma eficiente.

Simula IF

O Simula IF foi um projeto pensado para corrigir automaticamente os simulados aplicados em minha escola. Esse projeto iria corrigir automaticamente as questões de múltipla escolha, e gerar a nota dos alunos automaticamente. Facilitando esse processo de correção e minimizando os erros humanos e o tempo gasto com a correção manual. Esse projeto também teria portação tanto para Web quanto para Android.
